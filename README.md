## What's Included

- **Complete adventure** set in 1920s Arkham, Massachusetts — the classic Lovecraftian New England setting
- **5 scenes** with descriptions, skill opportunities, and Keeper notes
- **Full NPC statistics** for every named character
- **Monster statistics** — the Ghoul, with tactics, wound thresholds, and special abilities
- **4 player handouts** ready to print or read aloud
- **5 pre-generated investigators** with full CoC 7e stats, derived values, complete skill lists, equipment, and sanity tracks
- **Quick rules reference** — dice mechanics, combat, sanity rolls, and temporary insanity table — so the Keeper doesn't need to memorise the rulebook
- **Keeper tips** — pacing guide, what to do if players go off-track, how to run the sanity system for new players

## Requirements

- **Rulebook:** Call of Cthulhu, 7th Edition (Chaosium). The adventure is self-contained, but the core rulebook is useful for looking up edge cases.
- **Dice:** Standard RPG set — d4, d6, d8, d10 ×2 (one for tens, one for units = d100/percentile), d12, d20
- **Players:** 3–5 (plus the Keeper/DM)
- **Time:** 3–5 hours
